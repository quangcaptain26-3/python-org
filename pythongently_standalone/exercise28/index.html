Tương tự Bài tập #27 nhưng chỉ vẽ viền hình chữ nhật: góc dùng +, cạnh ngang -, cạnh dọc | (giống đường kẻ Bài tập #25).

Mô tả bài tập

Viết hàm drawBorder(width, height) vẽ viền hình chữ nhật. drawBorder(16, 4) in:

+--------------+

|              |

|              |

+--------------+

Kích thước bao gồm cả góc. Nếu width hoặc height < 2, không in gì.

Kiến thức cần có: toán tử Boolean, chuỗi, nối/nhân chuỗi, vòng lặp for, range()

Thiết kế lời giải

Ba phần: hàng trên, hàng giữa (lặp), hàng dưới (giống hàng trên). Hàng trên/dưới: '+' + '-' * (width - 2) + '+'. Hàng giữa: '|' + ' ' * (width - 2) + '|', lặp height - 2 lần.

Nhân chuỗi: '-' * (width - 2) tạo đủ dấu gạch ngang.

Trường hợp đặc biệt và lỗi thường gặp

Kích thước tối thiểu là 2×2. drawBorder(2, 2) in ++ hai dòng. Nhỏ hơn 2 thì không in.

Mẫu lời giải

Mẫu: https://invpy.com/borderdrawing-template.py. Lời giải: https://invpy.com/borderdrawing.py.

Giải thích chi tiết

Bước 1 — Kiểm tra: if width < 2 or height < 2: return.

Bước 2 — Hàng trên: print('+' + ('-' * (width - 2)) + '+') — hai góc chiếm 2 đơn vị chiều rộng.

Bước 3 — Hàng giữa: Vòng for i in range(height - 2): in '|' + (' ' * (width - 2)) + '|'.

Bước 4 — Hàng dưới: Lặp lại mã hàng trên.

Mã nguồn hoàn chỉnh

Chương trình Python đầy đủ:

def drawBorder(width, height):

    if width < 2 or height < 2:

        return

    print('+' + ('-' * (width - 2)) + '+')

    for i in range(height - 2):

        print('|' + (' ' * (width - 2)) + '|')

    print('+' + ('-' * (width - 2)) + '+')
