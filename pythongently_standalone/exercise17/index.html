Bài tập #17: Tung xúc xắc

rollDice(1)  → 6

Bài tập này dùng hàm sinh số ngẫu nhiên của Python để mô phỏng tung bất kỳ số lượng xúc xắc 6 mặt nào và trả về tổng điểm. Bài tập giới thiệu module random trong Python.

Mô tả bài tập

Viết hàm rollDice() với tham số numberOfDice — số xúc xắc 6 mặt cần tung. Hàm trả về tổng điểm của tất cả lần tung. Bạn phải import random và gọi random.randint().

Sao chép các assert sau. Vì kết quả ngẫu nhiên, ta kiểm tra giá trị nằm trong khoảng hợp lệ:

assert rollDice(0) == 0

assert rollDice(1000) != rollDice(1000)

for i in range(1000):

    assert 1 <= rollDice(1) <= 6

    assert 2 <= rollDice(2) <= 12

    assert 3 <= rollDice(3) <= 18

    assert 100 <= rollDice(100) <= 600

Có xác suất cực nhỏ assert rollDice(1000) != rollDice(1000) thất bại do hai lần tung trùng tổng. Nếu vậy, chạy lại kiểm tra.

Kiến thức cần có: import, module random, randint(), vòng lặp for, range(), toán tử gán cộng

Thiết kế lời giải

Khởi tạo biến tổng bằng 0. Dùng vòng lặp for lặp numberOfDice lần. Mỗi lần gọi random.randint(1, 6) và cộng vào tổng. Sau vòng lặp trả về tổng.

Gõ help(random.randint) trong shell tương tác để xem tài liệu.

Trường hợp đặc biệt và lỗi thường gặp

random.randint(1, 6) bao gồm cả hai đầu mút (khác range()). Tung 3 xúc xắc cho tổng từ 3 đến 18, không phải 1 đến 18. Phải tung từng viên rồi cộng, không dùng randint(2, 12) một lần vì phân phối không đều. Cần import random ở đầu file.

Giải thích chi tiết

Bước 1 — Import: import random để dùng random.randint().

Bước 2 — Khởi tạo tổng: total = 0 tích lũy điểm các lần tung.

Bước 3 — Vòng lặp: for i in range(numberOfDice): — khi numberOfDice là 0, vòng lặp không chạy, trả về 0.

Bước 4 — Tung xúc xắc: total += random.randint(1, 6) mô phỏng một mặt ngẫu nhiên từ 1 đến 6.

Bước 5 — Trả về: return total sau khi tung đủ số lần.

Mã nguồn hoàn chỉnh

Chương trình Python đầy đủ:

import random

 

def rollDice(numberOfDice):

    total = 0

    for i in range(numberOfDice):

        total += random.randint(1, 6)

    return total

 

assert rollDice(0) == 0

assert rollDice(1000) != rollDice(1000)

for i in range(1000):

    assert 1 <= rollDice(1) <= 6

    assert 2 <= rollDice(2) <= 12

    assert 3 <= rollDice(3) <= 18

    assert 100 <= rollDice(100) <= 600
