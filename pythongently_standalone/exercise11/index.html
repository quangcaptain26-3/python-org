Bài tập #11: Giờ, phút, giây

getHoursMinutesSeconds(90)  →  '1m 30s'

Website thường hiển thị thời gian tương đối như “3 ngày trước”. Bài tập chuyển số giây thành chuỗi giờ, phút, giây.

Mô tả bài tập

getHoursMinutesSeconds(totalSeconds) — nếu giờ/phút/giây bằng 0 thì không hiển thị (trả '10m' không phải '0h 10m 0s'). Ngoại lệ: getHoursMinutesSeconds(0) trả '0s'.

Các assert:

assert getHoursMinutesSeconds(30) == '30s'

assert getHoursMinutesSeconds(60) == '1m'

assert getHoursMinutesSeconds(90) == '1m 30s'

assert getHoursMinutesSeconds(3600) == '1h'

assert getHoursMinutesSeconds(3601) == '1h 1s'

assert getHoursMinutesSeconds(3661) == '1h 1m 1s'

assert getHoursMinutesSeconds(90042) == '25h 42s'

assert getHoursMinutesSeconds(0) == '0s'

 

Thử thách thêm: tách chu kỳ 24 giờ thành ngày (hậu tố “d”), ví dụ getHoursMinutesSeconds(90042) → '1d 1h 42s'.

Khái niệm tiên quyết: join(), append(), list, nối chuỗi, vòng while

Thiết kế lời giải

1 giờ = 3600 giây, 1 phút = 60 giây. Trừ 3600 lặp để đếm giờ, rồi trừ 60 lặp để đếm phút; phần còn lại là giây. Thêm vào list dạng '1h', '30s' chỉ khi > 0; ' '.join(hms) nối bằng dấu cách.

Giải thích chi tiết

if totalSeconds == 0: return '0s' — trường hợp đặc biệt.

Vòng while totalSeconds >= 3600 — trừ giờ trước, phút sau, để không nhầm 7200 giây thành 120 phút thay vì 2 giờ.

Chỉ append thành phần khác 0 (trừ 0 giây đã xử lý riêng).

Mã nguồn hoàn chỉnh

def getHoursMinutesSeconds(totalSeconds):

if totalSeconds == 0:

return '0s'

hours = 0

while totalSeconds >= 3600:

hours += 1

totalSeconds -= 3600

minutes = 0

while totalSeconds >= 60:

minutes += 1

totalSeconds -= 60

seconds = totalSeconds

hms = []

if hours > 0:

hms.append(str(hours) + 'h')

if minutes > 0:

hms.append(str(minutes) + 'm')

if seconds > 0:

hms.append(str(seconds) + 's')

return ' '.join(hms)

Trường hợp đặc biệt và lỗi thường gặp

Trừ đơn vị lớn trước (giờ → phút → giây). Không hiển thị thành phần 0.

Mẫu lời giải

Mã dở: https://invpy.com/hoursminutesseconds-template.py. Cách dùng // và %: https://invpy.com/hoursminutesseconds2.py.

Thiết kế lời giải thay thế

hours = totalSeconds // 3600, totalSeconds %= 3600; tương tự phút với 60.
