Bài tập #9: Màu ô cờ vua

getChessSquareColor(7, 7)  →  'white'
getChessSquareColor(1, 0)  →  'black'

Bàn cờ có ô trắng đen xen kẽ. Bài tập yêu cầu tìm quy luật màu theo cột và hàng, rồi chuyển thành mã Python — luyện mô hình hóa đối tượng thực tế.

Mô tả bài tập

Viết getChessSquareColor(column, row) trả về 'black' hoặc 'white'. Bàn 8×8; cột và hàng từ 0 đến 7 (Hình 9-1). Nếu column hoặc row ngoài 0–7, trả về chuỗi rỗng ''.

Góc trên trái luôn là ô trắng.

Hình 9-1: Bàn cờ với cột và hàng được đánh số.

Các assert:

assert getChessSquareColor(0, 0) == 'white'

assert getChessSquareColor(1, 0) == 'black'

assert getChessSquareColor(0, 1) == 'black'

assert getChessSquareColor(7, 7) == 'white'

assert getChessSquareColor(0, 8) == ''

assert getChessSquareColor(2, 9) == ''

Khái niệm tiên quyết: toán tử Boolean, modulo

Thiết kế lời giải

Cột và hàng cùng chẵn hoặc cùng lẻ → trắng; một chẵn một lẻ → đen. Dùng column % 2 và row % 2 so sánh với nhau.

Giải thích chi tiết

Kiểm tra biên trước: if not (0 <= column <= 7 and 0 <= row <= 7): return ''.

if column % 2 == row % 2: return 'white' — (0,0): 0==0 → trắng; (1,0): 1≠0 → đen.

Ngược lại return 'black'.

Mã nguồn hoàn chỉnh

def getChessSquareColor(column, row):

if not (0 <= column <= 7 and 0 <= row <= 7):

return ''

if column % 2 == row % 2:

return 'white'

else:

return 'black'

Trường hợp đặc biệt và lỗi thường gặp

Kiểm tra cột/hàng ngoài 0–7 trước khi tính modulo.
