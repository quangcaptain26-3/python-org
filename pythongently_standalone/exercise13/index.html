Bài tập #13: Tổng & Tích

  calculateSum([2, 4, 6, 8, 10])  →  30
calculateProduct([2, 4, 6, 8, 10])  →  3840

Hàm sum() có sẵn trong Python. Bài tập tự viết calculateSum() và calculateProduct().

Mô tả bài tập

Hai hàm cùng tham số numbers (list số). calculateSum() cộng dồn; list rỗng → 0. calculateProduct() nhân dồn; list rỗng → 1. Không gọi sum().

Các assert:

assert calculateSum([]) == 0

assert calculateSum([2, 4, 6, 8, 10]) == 30

assert calculateProduct([]) == 1

assert calculateProduct([2, 4, 6, 8, 10]) == 3840

Khái niệm tiên quyết: list, chỉ số, for, toán tử gán tăng cường

Thiết kế lời giải

calculateSum: result = 0, cộng từng số. calculateProduct: result = 1, nhân từng số.

Giải thích chi tiết

Không đặt tên biến sum — trùng hàm built-in sum().

Tích list rỗng là 1 (phần tử trung tính nhân — multiplicative identity): 0 × bất kỳ = 0 sai cho tích rỗng. Tổng list rỗng là 0 (additive identity).

result += number và result *= number — gán tăng cường.

Mã nguồn hoàn chỉnh

def calculateSum(numbers):

result = 0

for number in numbers:

result += number

return result

 

def calculateProduct(numbers):

result = 1

for number in numbers:

result *= number

return result

Trường hợp đặc biệt và lỗi thường gặp

Tránh ghi đè tên built-in: all, min, max, sum, open, v.v.

Khởi tạo tích bằng 1, không phải 0.
