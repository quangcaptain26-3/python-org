Bài tập #19: Tạo mật khẩu

generatePassword(12)  →  'v*f6uoklQJ!d'
generatePassword(12)  →  ' Yzkr(j2T$MsG'
generatePassword(16)  →  ' UVp7ow8T%5LZl1la'

Mật khẩu từ một từ tiếng Anh như “rosebud” dễ nhớ nhưng không an toàn. Tấn công từ điển (dictionary attack) là khi hacker thử lần lượt mọi từ trong từ điển. Mật khẩu ngẫu nhiên khó bị tấn công kiểu này hơn.

Mô tả bài tập

Viết hàm generatePassword() với tham số length (số ký tự). Vì lý do bảo mật, nếu length < 12 thì ép thành 12. Mật khẩu phải có ít nhất một chữ thường, một chữ hoa, một số và một ký tự đặc biệt từ ~!@#$%^&*()_+. Dùng module random.

Các assert kiểm tra:

assert len(generatePassword(8)) == 12

 

pw = generatePassword(14)

assert len(pw) == 14

hasLowercase = False

hasUppercase = False

hasNumber = False

hasSpecial = False

for character in pw:

    if character in LOWER_LETTERS:

        hasLowercase = True

    if character in UPPER_LETTERS:

        hasUppercase = True

    if character in NUMBERS:

        hasNumber = True

    if character in SPECIAL:

        hasSpecial = True

assert hasLowercase and hasUppercase and hasNumber and hasSpecial

Kiến thức cần có: import, module random, chuỗi, nối chuỗi, len(), append(), randint(), shuffle(), join()

Thiết kế lời giải

Tạo hằng số cho từng loại ký tự, nối thành ALL_CHARS. Trong hàm: ép length tối thiểu 12; khởi tạo password là list rỗng; thêm ngẫu nhiên một ký tự từ mỗi loại; tiếp tục thêm từ ALL_CHARS đến đủ độ dài; gọi random.shuffle(password); trả về ''.join(password).

Trường hợp đặc biệt và lỗi thường gặp

random.shuffle() chỉ hoạt động với list, không phải chuỗi. Dùng list rồi join() để ghép.

Mẫu lời giải

Mẫu: https://invpy.com/passwordgenerator-template.py. Lời giải: https://invpy.com/passwordgenerator.py.

Đọc thêm

Hầu hết bộ sinh số ngẫu nhiên là giả ngẫu nhiên (pseudorandom): cùng random.seed() cho cùng dãy số. Python cũng có random.SystemRandom() dựa trên entropy hệ thống. Ứng dụng tạo mật khẩu này đủ tốt cho hầu hết người dùng. Nên dùng trình quản lý mật khẩu như KeePass (https://keepass.info).

Giải thích chi tiết

Bước 1 — Hằng số: Định nghĩa LOWER_LETTERS, UPPER_LETTERS, NUMBERS, SPECIAL và ALL_CHARS ở phạm vi toàn cục.

Bước 2 — Độ dài tối thiểu: if length < 12: length = 12.

Bước 3 — Bốn ký tự bắt buộc: Dùng random.randint() chọn chỉ số ngẫu nhiên trong mỗi chuỗi, append vào list.

Bước 4 — Đủ độ dài: Vòng while len(password) < length thêm ký tự từ ALL_CHARS.

Bước 5 — Xáo trộn và ghép: random.shuffle(password) rồi return ''.join(password).

Mã nguồn hoàn chỉnh

Chương trình Python đầy đủ:

import random

 

LOWER_LETTERS = 'abcdefghijklmnopqrstuvwxyz'

UPPER_LETTERS = 'ABCDEFGHIJKLMNOPQRSTUVWXYZ'

NUMBERS = '1234567890'

SPECIAL = '~!@#$%^&*()_+'

ALL_CHARS = LOWER_LETTERS + UPPER_LETTERS + NUMBERS + SPECIAL

 

def generatePassword(length):

    if length < 12:

        length = 12

    password = []

    password.append(LOWER_LETTERS[random.randint(0, 25)])

    password.append(UPPER_LETTERS[random.randint(0, 25)])

    password.append(NUMBERS[random.randint(0, 9)])

    password.append(SPECIAL[random.randint(0, 12)])

    while len(password) < length:

        password.append(ALL_CHARS[random.randint(0, 74)])

    random.shuffle(password)

    return ''.join(password)

 

assert len(generatePassword(8)) == 12

 

pw = generatePassword(14)

assert len(pw) == 14

hasLowercase = False

hasUppercase = False

hasNumber = False

hasSpecial = False

for character in pw:

    if character in LOWER_LETTERS:

        hasLowercase = True

    if character in UPPER_LETTERS:

        hasUppercase = True

    if character in NUMBERS:

        hasNumber = True

    if character in SPECIAL:

        hasSpecial = True

assert hasLowercase and hasUppercase and hasNumber and hasSpecial
