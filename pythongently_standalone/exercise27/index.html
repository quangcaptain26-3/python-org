Bài tập tạo nghệ thuật ASCII — đồ họa từ ký tự văn bản. Ở đây vẽ hình chữ nhật đặc bằng ký tự #.

Mô tả bài tập

Viết hàm drawRectangle(width, height) — hai tham số nguyên. Hàm không trả về giá trị mà in hình chữ nhật gồm width ký tự # theo chiều ngang và height hàng.

Nếu width hoặc height là 0 hoặc số âm, không in gì.

Kiến thức cần có: vòng lặp for, range(), print(), tham số end

Thiết kế lời giải

Hai vòng lặp lồng nhau: trong in một hàng width ký tự # với end=''; ngoài lặp height lần. Hoặc dùng nhân chuỗi: '#' * width mỗi hàng — gọn hơn.

Ví dụ: '#' * 16 → '################'.

Trường hợp đặc biệt và lỗi thường gặp

Trong terminal, ký tự chữ cao gấp đôi rộng — muốn hình vuông trực quan thì width nên gấp đôi height.

Giải thích chi tiết

Bước 1 — Kiểm tra đầu vào: if width < 1 or height < 1: return.

Bước 2 — Vòng hàng: for row in range(height):.

Bước 3a — Cách lồng vòng: Vòng cột in print('#', end=''), sau hàng gọi print().

Bước 3b — Cách nhân chuỗi: print('#' * width) mỗi hàng — đơn giản hơn.

Mã nguồn hoàn chỉnh

Chương trình Python đầy đủ (dùng nhân chuỗi):

def drawRectangle(width, height):

    if width < 1 or height < 1:

        return

    for row in range(height):

        print('#' * width)
