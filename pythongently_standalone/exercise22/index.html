Bài tập #22: Kéo búa bao

rpsWinner('rock', 'paper')  →  'player two'

Kéo búa bao là trò chơi phổ biến cho hai người. Hai người cùng chọn kéo, búa hoặc bao: búa thắng kéo, bao thắng búa, kéo thắng bao. Bài tập xác định người thắng từ nước đi của hai người chơi.

Mô tả bài tập

Viết hàm rpsWinner(player1, player2). Mỗi tham số là 'rock', 'paper' hoặc 'scissors'. Người 1 thắng trả về 'player one', người 2 thắng trả về 'player two', hòa trả về 'tie'.

Các assert kiểm tra:

assert rpsWinner('rock', 'paper') == 'player two'

assert rpsWinner('rock', 'scissors') == 'player one'

assert rpsWinner('paper', 'scissors') == 'player two'

assert rpsWinner('paper', 'rock') == 'player one'

assert rpsWinner('scissors', 'rock') == 'player two'

assert rpsWinner('scissors', 'paper') == 'player one'

assert rpsWinner('rock', 'rock') == 'tie'

assert rpsWinner('paper', 'paper') == 'tie'

assert rpsWinner('scissors', 'scissors') == 'tie'

Kiến thức cần có: toán tử Boolean, elif

Thiết kế lời giải

Dùng chuỗi if-elif-else. Kết hợp điều kiện bằng and, ví dụ player1 == 'rock' and player2 == 'paper' → người 2 thắng.

Trường hợp đặc biệt và lỗi thường gặp

Có thể kiểm tra player1 == player2 đầu hàm và trả về 'tie' ngay.

Mẫu lời giải

Mẫu: https://invpy.com/rockpaperscissors-template.py. Lời giải: https://invpy.com/rockpaperscissors.py.

Giải thích chi tiết

Bước 1: Liệt kê 6 tổ hợp có người thắng (không hòa), mỗi tổ hợp một nhánh elif.

Bước 2: Mỗi nhánh trả về 'player one' hoặc 'player two' theo quy tắc: búa>kéo, bao>búa, kéo>bao.

Bước 3: Nhánh else cuối cùng trả về 'tie' khi cùng nước đi.

Mã nguồn hoàn chỉnh

Chương trình Python đầy đủ:

def rpsWinner(player1, player2):

    if player1 == 'rock' and player2 == 'paper':

        return 'player two'

    elif player1 == 'rock' and player2 == 'scissors':

        return 'player one'

    elif player1 == 'paper' and player2 == 'scissors':

        return 'player two'

    elif player1 == 'paper' and player2 == 'rock':

        return 'player one'

    elif player1 == 'scissors' and player2 == 'rock':

        return 'player two'

    elif player1 == 'scissors' and player2 == 'paper':

        return 'player one'

    else:

        return 'tie'

 

assert rpsWinner('rock', 'paper') == 'player two'

assert rpsWinner('rock', 'scissors') == 'player one'

assert rpsWinner('paper', 'scissors') == 'player two'

assert rpsWinner('paper', 'rock') == 'player one'

assert rpsWinner('scissors', 'rock') == 'player two'

assert rpsWinner('scissors', 'paper') == 'player one'

assert rpsWinner('rock', 'rock') == 'tie'

assert rpsWinner('paper', 'paper') == 'tie'

assert rpsWinner('scissors', 'scissors') == 'tie'
