Bài tập #21: Kiểm tra ngày hợp lệ

isValidDate(2005, 3, 29)  →  True
isValidDate(2005, 13, 32)  →  False

Ba số nguyên năm/tháng/ngày không phải lúc nào cũng tạo thành ngày hợp lệ — không có tháng 13 hay ngày 32. Bài tập kiểm tra tổ hợp có hợp lệ không, tùy số ngày từng tháng. Cần hoàn thành Bài tập #20 trước để dùng isLeapYear() cho ngày 29/2.

Mô tả bài tập

Viết hàm isValidDate(year, month, day). Trả về True nếu hợp lệ, False nếu không. Tháng từ 1 đến 12. Ngày từ 1 đến 28, 29, 30 hoặc 31 tùy tháng. import leapyear từ file leapyear.py Bài tập #20.

Tháng 9, 4, 6, 11 có 30 ngày. Các tháng còn lại 31 ngày, trừ tháng 2 (28 ngày, 29 nếu năm nhuận).

Các assert kiểm tra:

assert isValidDate(1999, 12, 31) == True

assert isValidDate(2000, 2, 29) == True

assert isValidDate(2001, 2, 29) == False

assert isValidDate(2029, 13, 1) == False

assert isValidDate(1000000, 1, 1) == True

assert isValidDate(2015, 4, 31) == False

assert isValidDate(1970, 5, 99) == False

assert isValidDate(1981, 0, 3) == False

assert isValidDate(1666, 4, 0) == False

 

import datetime

d = datetime.date(1970, 1, 1)

oneDay = datetime.timedelta(days=1)

for i in range(1000000):

    assert isValidDate(d.year, d.month, d.day) == True

    d += oneDay

Kiến thức cần có: import, toán tử Boolean, chuỗi điều kiện, elif

Thiết kế lời giải

Mọi số nguyên đều hợp lệ cho year, nhưng cần leapyear.isLeapYear(year) khi month == 2. Kiểm tra 1 <= month <= 12 (có thể viết gọn). Số ngày tối đa: tháng 4,6,9,11 → 30; tháng 2 → 28 (29 nếu nhuận); còn lại → 31.

Trường hợp đặc biệt và lỗi thường gặp

Trả về False sớm nếu tháng ngoài 1–12. Nếu năm nhuận và 29/2 thì trả về True ngay.

Giải thích chi tiết

Bước 1 — Kiểm tra tháng: if not (1 <= month <= 12): return False.

Bước 2 — Ngày 29/2 năm nhuận: if leapyear.isLeapYear(year) and month == 2 and day == 29: return True.

Bước 3 — Tháng 31 ngày: Tháng 1,3,5,7,8,10,12 — ngày phải từ 1 đến 31.

Bước 4 — Tháng 30 ngày: Tháng 4,6,9,11 — ngày từ 1 đến 30.

Bước 5 — Tháng 2: Ngày từ 1 đến 28 (đã xử lý 29 ở bước 2).

Bước 6: Nếu qua hết kiểm tra, return True.

Mã nguồn hoàn chỉnh

Chương trình Python đầy đủ:

import leapyear

 

def isValidDate(year, month, day):

    if not (1 <= month <= 12):

        return False

    if leapyear.isLeapYear(year) and month == 2 and day == 29:

        return True

    if month in (1, 3, 5, 7, 8, 10, 12) and not (1 <= day <= 31):

        return False

    elif month in (4, 6, 9, 11) and not (1 <= day <= 30):

        return False

    elif month == 2 and not (1 <= day <= 28):

        return False

    return True

 

assert isValidDate(1999, 12, 31) == True

assert isValidDate(2000, 2, 29) == True

assert isValidDate(2001, 2, 29) == False

assert isValidDate(2029, 13, 1) == False

assert isValidDate(1000000, 1, 1) == True

assert isValidDate(2015, 4, 31) == False

assert isValidDate(1970, 5, 99) == False

assert isValidDate(1981, 0, 3) == False

assert isValidDate(1666, 4, 0) == False
