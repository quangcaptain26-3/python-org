Bài tập #6: Hậu tố thứ tự

ordinalSuffix(42)  →  '42nd'

Số đếm (cardinal) chỉ số lượng như “bốn quả táo”; số thứ tự (ordinal) chỉ vị trí trong dãy như “hạng nhất” hay “sinh nhật lần thứ 30”. Bài tập xử lý chuỗi nhiều hơn là toán học.

Mô tả bài tập

Trong tiếng Anh, số thứ tự có hậu tố như “th” trong “30th” hay “nd” trong “2nd”. Viết ordinalSuffix() với tham số nguyên number, trả về chuỗi số kèm hậu tố. Ví dụ ordinalSuffix(42) trả về '42nd'.

Có thể dùng str() để đổi số thành chuỗi. Không dùng endswith() trong lời giải (để giữ thử thách).

Các assert kiểm tra:

assert ordinalSuffix(0) == '0th'

assert ordinalSuffix(1) == '1st'

assert ordinalSuffix(2) == '2nd'

assert ordinalSuffix(3) == '3rd'

assert ordinalSuffix(4) == '4th'

assert ordinalSuffix(10) == '10th'

assert ordinalSuffix(11) == '11th'

assert ordinalSuffix(12) == '12th'

assert ordinalSuffix(13) == '13th'

assert ordinalSuffix(14) == '14th'

assert ordinalSuffix(101) == '101st'

Khái niệm tiên quyết: string, toán tử in, slice, nối chuỗi

Thiết kế lời giải

Hậu tố phụ thuộc hai chữ số cuối:

·       Hai chữ số cuối là 11, 12, 13 → “th”

·       Kết thúc bằng 1 → “st”; 2 → “nd”; 3 → “rd”

·       Còn lại → “th”

Dùng str(number) → numberStr. numberStr[-1] là chữ số cuối; numberStr[-2:] là hai chữ số cuối. So sánh nhiều giá trị: numberStr[-2:] in ('11', '12', '13').

Giải thích chi tiết

numberStr = str(number) — chuyển số sang string để dùng slice và so sánh ký tự.

Kiểm tra 11/12/13 trước — nếu không, 213 sẽ nhận “rd” thay vì “th”.

numberStr[-1] == '1' — chỉ số âm đếm từ cuối chuỗi; nối + 'st' để có hậu tố tiếng Anh.

Cách khác: number % 100 in (11, 12, 13) và number % 10 cho chữ số cuối (xem “Thiết kế lời giải thay thế”).

Mã nguồn hoàn chỉnh

def ordinalSuffix(number):

numberStr = str(number)

if numberStr[-2:] in ('11', '12', '13'):

return numberStr + 'th'

if numberStr[-1] == '1':

return numberStr + 'st'

if numberStr[-1] == '2':

return numberStr + 'nd'

if numberStr[-1] == '3':

return numberStr + 'rd'

return numberStr + 'th'

Trường hợp đặc biệt và lỗi thường gặp

Kiểm tra 11, 12, 13 trước khi kiểm tra 1, 2, 3. Số 0 có hậu tố “th”: '0th'.

Mẫu lời giải

Mã dở từ https://invpy.com/ordinalsuffix-template.py.

Lời giải đầy đủ: Phụ lục A, https://invpy.com/ordinalsuffix.py. Cách dùng modulo: https://invpy.com/ordinalsuffix2.py.

Thiết kế lời giải thay thế

Dùng modulo: number % 10 là chữ số hàng đơn vị; number % 100 là hai chữ số cuối. Ví dụ: 42 % 10 → 2; 12345 % 100 → 45.
