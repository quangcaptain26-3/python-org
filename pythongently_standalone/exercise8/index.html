Bài tập #8: Đọc ghi file

File I/O (nhập/xuất file) cho phép chương trình đọc và ghi dữ liệu lên ổ đĩa. Bài tập này ghi và đọc văn bản — quan trọng vì dữ liệu không mất khi chương trình đóng.

Mô tả bài tập

Viết ba hàm: writeToFile(filename, text) — ghi đè file; appendToFile(filename, text) — ghi thêm cuối file; readFromFile(filename) — trả về toàn bộ nội dung dạng string.

Kiểm tra:

writeToFile('greet.txt', 'Hello!\n')

appendToFile('greet.txt', 'Goodbye!\n')

assert readFromFile('greet.txt') == 'Hello!\nGoodbye!\n'

Sau khi chạy có thể xóa file greet.txt.

Khái niệm tiên quyết: đọc ghi file văn bản

Thiết kế lời giải

open(filename, mode) với with: 'w' ghi, 'a' ghi thêm, 'r' đọc (mặc định). fileObj.write(text) ghi; fileObj.read() đọc hết.

Giải thích chi tiết

with open(...) as fileObj: — tự đóng file khi ra khỏi khối, kể cả khi lỗi.

'w' tạo file mới hoặc xóa nội dung cũ. 'a' giữ nội dung và thêm vào cuối — nên Hello! rồi Goodbye! nối liền.

\n trong chuỗi là ký tự xuống dòng — assert so khớp cả ký tự này.

Mã nguồn hoàn chỉnh

def writeToFile(filename, text):

with open(filename, 'w') as fileObj:

fileObj.write(text)

 

def appendToFile(filename, text):

with open(filename, 'a') as fileObj:

fileObj.write(text)

 

def readFromFile(filename):

with open(filename) as fileObj:

return fileObj.read()

Trường hợp đặc biệt và lỗi thường gặp

Chế độ 'w' ghi đè toàn bộ file. 'a' không xóa nội dung cũ.
