Bài tập #14: Trung bình

average([12, 20, 37])  →  23

Trung bình (average) là tổng các số chia cho số lượng. Ví dụ: (12+1+5)/3 = 6. Ba bài thống kê tiếp theo luyện tính toán này bằng Python.

Mô tả bài tập

average(numbers) trả về trung bình cộng. Thử không dùng sum() built-in. List rỗng → None.

Các assert:

assert average([1, 2, 3]) == 2

assert average([1, 2, 3, 1, 2, 3, 1, 2, 3]) == 2

assert average([12, 20, 37]) == 23

assert average([0, 0, 0, 0, 0]) == 0

import random

random.seed(42)

testData = [1, 2, 3, 1, 2, 3, 1, 2, 3]

for i in range(1000):

    random.shuffle(testData)

    assert average(testData) == 2

Xáo trộn thứ tự không đổi trung bình. random.seed(42) — xem Bài #19.

Khái niệm tiên quyết: len(), for, toán tử gán tăng cường

Thiết kế lời giải

Giống calculateSum() Bài #13: cộng dồn vào total, rồi return total / len(numbers). Có thể import sumproduct và gọi sumproduct.calculateSum().

Giải thích chi tiết

Kiểm tra list rỗng đầu hàm — tránh chia cho 0.

total = 0, vòng for cộng từng phần tử.

return total / len(numbers) — phép chia có thể trả float ngay cả khi chia hết (Python 3).

Mã nguồn hoàn chỉnh

def average(numbers):

if len(numbers) == 0:

return None

total = 0

for number in numbers:

total += number

return total / len(numbers)

Trường hợp đặc biệt và lỗi thường gặp

List rỗng → None, không chia cho len([]) (= 0).
