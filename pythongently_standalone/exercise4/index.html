Bài tập #4: Diện tích & Thể tích

         area(10, 4)  →  40
    perimeter(10, 4)  →  28
     volume(10, 4, 5)  →  200
surfaceArea(10, 4, 5)  →  220

Diện tích, chu vi, thể tích và diện tích toàn phần là các phép tính trực tiếp. Bài này giống Bài #2 và #3: mỗi hàm là một phép tính và return. Khác biệt là nhiều tham số hơn — luyện chuyển công thức toán sang Python.

Mô tả bài tập

Viết bốn hàm. area() và perimeter() có tham số length và width; volume() và surfaceArea() có thêm height. Các hàm trả về diện tích, chu vi, thể tích và diện tích toàn phần tương ứng.

Công thức theo chiều dài (L), rộng (W), cao (H):

·       diện tích = L × W

·       chu vi = L + W + L + W

·       thể tích = L × W × H

·       diện tích toàn phần = (L × W × 2) + (L × H × 2) + (W × H × 2)

Diện tích là đo hai chiều (dài × rộng). Chu vi là tổng bốn cạnh. Thể tích là đo ba chiều. Diện tích toàn phần là tổng sáu mặt hình chữ nhật của hình hộp chữ nhật.

Xem Hình 4-1 minh họa các công thức.

Hình 4-1: Các thành phần diện tích, thể tích, chu vi và diện tích toàn phần.

Sao chép các assert sau. Lời giải đúng khi tất cả là True:

assert area(10, 10) == 100

assert area(0, 9999) == 0

assert area(5, 8) == 40

assert perimeter(10, 10) == 40

assert perimeter(0, 9999) == 19998

assert perimeter(5, 8) == 26

assert volume(10, 10, 10) == 1000

assert volume(9999, 0, 9999) == 0

assert volume(5, 8, 10) == 400

assert surfaceArea(10, 10, 10) == 600

assert surfaceArea(9999, 0, 9999) == 199960002

assert surfaceArea(5, 8, 10) == 340

Khái niệm tiên quyết: toán tử toán học, thứ tự ưu tiên toán tử

Thiết kế lời giải

Giống Bài #2: thay L, W, H bằng length, width, height; thay + và × bằng + và *. Mỗi hàm một dòng return.

Giải thích chi tiết

area(length, width) — return length * width: nhân hai kích thước mặt đáy hình chữ nhật.

perimeter(length, width) — return length * 2 + width * 2 hoặc tương đương 2 * (length + width): tổng bốn cạnh.

volume(length, width, height) — nhân ba chiều của hình hộp chữ nhật.

surfaceArea(...) — ba cặp mặt đối diện: L×W, L×H, W×H; mỗi cặp có hai mặt nên nhân tổng với 2. Công thức trong sách: ((length * width) + (length * height) + (width * height)) * 2.

Mã nguồn hoàn chỉnh

def area(length, width):

return length * width

 

def perimeter(length, width):

return length * 2 + width * 2

 

def volume(length, width, height):

return length * width * height

 

def surfaceArea(length, width, height):

return ((length * width) + (length * height) + (width * height)) * 2

Trường hợp đặc biệt và lỗi thường gặp

Chiều dài, rộng, cao thường là số dương. Có thể kiểm tra số âm và ném exception, nhưng bài tập không yêu cầu.
