Bài tập #12: Nhỏ nhất & Lớn nhất

getSmallest([28, 25, 42, 2, 28])  →  2

Hàm min() và max() có sẵn trong Python. Bài tập yêu cầu tự viết lại — không dùng min().

Mô tả bài tập

getSmallest(numbers) — list số nguyên/thực; trả số nhỏ nhất; list rỗng → None. Sau đó viết getBiggest() trả số lớn nhất.

Các assert:

assert getSmallest([1, 2, 3]) == 1

assert getSmallest([3, 2, 1]) == 1

assert getSmallest([28, 25, 42, 2, 28]) == 2

assert getSmallest([1]) == 1

assert getSmallest([]) == None

Khái niệm tiên quyết: len(), for, list, None

Thiết kế lời giải

Gán smallest = numbers[0], lặp từng phần tử; nếu nhỏ hơn smallest thì cập nhật. Hình 12-1 minh họa với [28, 25, 42, 2, 28].

Hình 12-1: Biến smallest trong vòng for trên list [28, 25, 42, 2, 28].

Giải thích chi tiết

if len(numbers) == 0: return None — list rỗng không có phần tử để so sánh.

Khởi tạo từ phần tử đầu đảm bảo smallest luôn thuộc list.

getBiggest: cùng logic với > thay <.

Mã nguồn hoàn chỉnh

def getSmallest(numbers):

if len(numbers) == 0:

return None

smallest = numbers[0]

for number in numbers:

if number < smallest:

smallest = number

return smallest

 

def getBiggest(numbers):

if len(numbers) == 0:

return None

biggest = numbers[0]

for number in numbers:

if number > biggest:

biggest = number

return biggest

Trường hợp đặc biệt và lỗi thường gặp

List rỗng → None ngay, không truy cập numbers[0].
