Bài tập #16: Mode (Số xuất hiện nhiều nhất)

mode([1, 1, 2, 3, 4])  →  1

Mode là bài tập thống kê thứ ba trong sách. Mode là số xuất hiện thường xuyên nhất trong danh sách. Cùng với trung vị và trung bình, bạn có bộ chỉ số mô tả tổng quan cho một tập số. Bài tập này kiểm tra khả năng dùng từ điển (dictionary) để đếm tần suất các số.

Mô tả bài tập

Viết hàm mode() nhận tham số numbers. Hàm trả về mode — số xuất hiện nhiều nhất — trong danh sách số nguyên và số thực.

Truyền danh sách rỗng vào mode() phải trả về None.

Sao chép các câu lệnh assert sau vào cuối chương trình:

assert mode([]) == None

assert mode([1, 2, 3, 4, 4]) == 4

assert mode([1, 1, 2, 3, 4]) == 1

import random

random.seed(42)

testData = [1, 2, 3, 4, 4]

for i in range(1000):

    random.shuffle(testData)

    assert mode(testData) == 4

Xáo trộn thứ tự không ảnh hưởng mode. Vòng lặp kiểm tra 1.000 lần. Giải thích random.seed() xem Bài tập #19.

Hãy tự viết lời giải trước. Nếu gặp khó khăn, đọc Thiết kế lời giải và Trường hợp đặc biệt và lỗi thường gặp.

Kiến thức cần có: vòng lặp for, toán tử gán cộng, chỉ số, toán tử not in

Thiết kế lời giải

Dùng từ điển để đếm tần suất: khóa là số, giá trị là số lần xuất hiện. Khởi tạo từ điển rỗng và hai biến theo dõi số xuất hiện nhiều nhất cùng số lần xuất hiện của nó. Duyệt từng số: nếu chưa có trong từ điển thì tạo khóa với giá trị 0, rồi tăng đếm. Nếu đếm mới lớn hơn kỷ lục, cập nhật số xuất hiện nhiều nhất.

Cách làm tương tự Bài tập #12 “Nhỏ nhất & Lớn nhất”: duyệt danh sách và cập nhật biến theo dõi kết quả tốt nhất tìm được.

Trường hợp đặc biệt và lỗi thường gặp

Danh sách rỗng phải trả về None ngay từ đầu hàm.

Mẫu lời giải

Mẫu tại https://invpy.com/mode-template.py. Lời giải đầy đủ: https://invpy.com/mode.py, debugger: https://invpy.com/mode-debug/.

Giải thích chi tiết

Bước 1 — Danh sách rỗng: Trả về None nếu len(numbers) == 0.

Bước 2 — Khởi tạo: numberCount = {} lưu tần suất; mostFreqNumber và mostFreqNumberCount = 0 theo dõi kỷ lục.

Bước 3 — Đếm: Với mỗi number, dùng if number not in numberCount để khởi tạo khóa, rồi numberCount[number] += 1.

Bước 4 — Cập nhật kỷ lục: Nếu numberCount[number] > mostFreqNumberCount, gán lại cả hai biến theo dõi.

Bước 5 — Trả về: Sau vòng lặp, mostFreqNumber chính là mode.

Mã nguồn hoàn chỉnh

Chương trình Python đầy đủ:

def mode(numbers):

    if len(numbers) == 0:

        return None

    numberCount = {}

    mostFreqNumber = None

    mostFreqNumberCount = 0

    for number in numbers:

        if number not in numberCount:

            numberCount[number] = 0

        numberCount[number] += 1

        if numberCount[number] > mostFreqNumberCount:

            mostFreqNumber = number

            mostFreqNumberCount = numberCount[number]

    return mostFreqNumber

 

assert mode([]) == None

assert mode([1, 2, 3, 4, 4]) == 4

assert mode([1, 1, 2, 3, 4]) == 1

import random

random.seed(42)

testData = [1, 2, 3, 4, 4]

for i in range(1000):

    random.shuffle(testData)

    assert mode(testData) == 4
