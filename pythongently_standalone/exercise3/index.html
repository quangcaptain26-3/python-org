Bài tập #3: Chẵn & Lẻ

isOdd(13)  →  True
isEven(13)  →  False

Xác định số chẵn hay lẻ là phép tính phổ biến dùng toán tử modulo. Giống Bài tập #2, mỗi hàm trong lời giải có thể chỉ một dòng.

Bài tập giới thiệu toán tử % (modulo) và kỹ thuật modulo-2 để biết số chẵn hay lẻ.

Mô tả bài tập

Viết hai hàm isOdd() và isEven(), mỗi hàm có một tham số số number. isOdd() trả về True nếu number lẻ, False nếu chẵn. isEven() trả về True nếu chẵn, False nếu lẻ. Cả hai trả về False với số có phần thập phân như 3.14 hoặc -4.5. Số 0 được coi là chẵn.

Sao chép các assert sau xuống cuối chương trình. Lời giải đúng khi tất cả điều kiện là True:

assert isOdd(42) == False

assert isOdd(9999) == True

assert isOdd(-10) == False

assert isOdd(-11) == True

assert isOdd(3.1415) == False

assert isEven(42) == True

assert isEven(9999) == False

assert isEven(-10) == True

assert isEven(-11) == False

assert isEven(3.1415) == False

Khái niệm tiên quyết: toán tử modulo

Thiết kế lời giải

Bài này dễ nếu đã biết cách làm, khó nếu phải tự nghĩ từ đầu. Hãy tra cứu trên mạng kèm từ “Python”, ví dụ “python kiểm tra số chẵn lẻ”. Stack Overflow (https://stackoverflow.com) thường có câu trả lời tốt. Tra cứu trực tuyến không phải “gian lận” — lập trình viên chuyên nghiệp tra cứu hàng chục lần mỗi ngày!

Toán tử % chia lấy dư. Chia number cho 2: dư 1 → lẻ; dư 0 → chẵn. Ví dụ: 42 % 2 là 0 (chẵn); 7 % 2 là 1 (lẻ).

Số thực như 3.1415 không chẵn cũng không lẻ, nên cả isOdd() và isEven() trả về False.

Hàm phải trả về Boolean True hoặc False, không phải số 0 hay 1. Dùng == hoặc !=: 7 % 2 == 1 → True.

Giải thích chi tiết

return number % 2 == 1 trong isOdd() — number % 2 là phần dư khi chia cho 2. So sánh với 1: đúng thì lẻ. Với số thực như 3.1415, 3.1415 % 2 không bằng 0 hay 1 nguyên nên điều kiện sai → trả về False (đúng yêu cầu).

return number % 2 == 0 trong isEven() — dư 0 nghĩa là chia hết cho 2 → chẵn. Số âm vẫn hoạt động: -10 % 2 == 0.

Hai hàm đối ngược nhau về logic nhưng không gọi lẫn nhau — mỗi hàm tự kiểm tra modulo.

Mã nguồn hoàn chỉnh

def isOdd(number):

return number % 2 == 1

 

def isEven(number):

return number % 2 == 0

Số không nguyên như 4.5 hay 3.1415 không chẵn cũng không lẻ — cả hai hàm trả về False.
